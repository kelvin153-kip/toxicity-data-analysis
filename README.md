Toxicity Classification Model
Overview

This project applies machine learning techniques to predict chemical toxicity using molecular descriptor features. The pipeline includes EDA, preprocessing, feature reduction, SMOTE balancing, and an ensemble learning model.

Dataset

Molecular descriptor dataset

Target column: Class

Labels: Toxic / NonToxic

Workflow

Data exploration and visualization

Missing value handling

Feature preprocessing

SMOTE class balancing

Feature reduction

Ensemble model training

Cross-validation and performance evaluation

Tools

Python, Pandas, Matplotlib, Scikit-learn, Imbalanced-learn
